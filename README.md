# QA Playwright E2E Tests

Projeto de Automação de Testes End-to-End (E2E) utilizando [Playwright](https://playwright.dev/) e TypeScript.

## 🚀 Tecnologias Utilizadas

* **Playwright:** Framework de automação E2E.
* **TypeScript:** Tipagem estática para maior segurança no código.
* **Faker.js:** Geração de dados de massa dinâmicos para testes.
* **commit-and-tag-version:** Versionamento automatizado e geração de CHANGELOG.

## 📦 Instalação

Clone o repositório e instale as dependências:

```bash
npm install
npx playwright install
